# HealthSync — 100% Free Cloud Deployment Guide 🚀

This guide provides a step-by-step walkthrough to deploy **HealthSync** entirely for free using:
1.  **Clever Cloud**: Provides a 100% free, 24/7 online MySQL database (no credit card required).
2.  **Render**: Hosts your Tomcat Docker container for free directly from your GitHub repository.

---

## 🗄️ Step 1: Create your Free MySQL Database on Clever Cloud

Clever Cloud offers a free shared MySQL database add-on that is perfect for school and personal projects.

### 1.1: Provision the MySQL Database
1. Go to [Clever Cloud](https://www.clever-cloud.com/) and register for a free account.
2. In the console, click the **Create** button in the top right and select **an add-on**.
3. Select **MySQL** from the list of database add-ons.
4. On the plan selection screen, choose the **Free** tier (this is a shared MySQL plan that costs $0/month and requires no credit card).
5. Click **Next** to deploy it. It will be ready instantly.
6. Click on your newly created MySQL add-on in your dashboard to view the **Connection details**:
   *   **Host**: `xxxxxx-mysql.services.clever-cloud.com`
   *   **Port**: `3306`
   *   **Database Name**: *(Generated by Clever Cloud, starts with `b`)*
   *   **Username**: *(Generated by Clever Cloud, starts with `u`)*
   *   **Password**: *(Generated by Clever Cloud password)*

### 1.2: Import the Database Schema
1. On your Clever Cloud MySQL dashboard page, click on the **phpMyAdmin** link on the left menu (this is a built-in web browser database editor).
2. On the phpMyAdmin sidebar, click on your database name (starts with `b`).
3. Click the **Import** tab in the top menu bar.
4. Click **Choose File** and select your local [schema.sql](file:///Users/ameypatil/Desktop/HealthSync/schema.sql) file.
5. Scroll down to the bottom and click **Go**.
6. **Success!** Your database tables (`users`, `vitals`, `allergies`, `medical_records`, and `prescriptions`) are now fully created in the cloud.

## 🌐 Step 2: Deploy the Tomcat Web App on Render (Free Tier)

Render's Web Service free tier fully supports custom Docker deployments using the `Dockerfile` we already set up.

### 2.1: Create the Web Service
1. Log in to [Render.com](https://render.com/).
2. Click **New +** and select **Web Service**.
3. Select your connected GitHub Repository: `AmeyPatil3/HealthSync`.
4. Configure the service settings:
   * **Name**: `healthsync-web`
   * **Region**: *Select a region close to Europe or US East (e.g. Frankfurt or Ohio) to match Clever Cloud's servers for fast response times.*
   * **Runtime**: Automatically set to **Docker** (based on your root `Dockerfile`).
   * **Instance Type**: Select **Free** (0.5 CPU, 512 MB RAM).

### 2.2: Add Environment Variables
Scroll down, click **Advanced**, and add your Clever Cloud MySQL database connection variables:

| Key | Value | Description |
| :--- | :--- | :--- |
| `DB_HOST` | *(Your Clever Cloud Host)* | e.g. `xxxxxx-mysql.services.clever-cloud.com` |
| `DB_PORT` | `3306` | Default MySQL port |
| `DB_NAME` | *(Your Clever Cloud Database Name)* | e.g. `bxxxxxxxxxxxxxxxxxx` |
| `DB_USER` | *(Your Clever Cloud Username)* | e.g. `uxxxxxxxxxxxxxxxxxx` |
| `DB_PASSWORD` | *(Your Clever Cloud Password)* | e.g. `xxxxxxxxxxxxxxxxxxxx` |

### 2.3: Build & Deploy
1. Click **Create Web Service**.
2. Render will pull your code from GitHub, read the `Dockerfile`, package the JSP pages and Tomcat server, and start the app.
3. Once the build log says **Live**, you will find your public URL at the top left of the dashboard (e.g. `https://healthsync-web.onrender.com`).
4. Point your browser to your deployed path:
   ```text
   https://YOUR_SUBDOMAIN.onrender.com/newhealth/index.jsp
   ```

## 💡 Important Free-Tier Details to Keep in Mind:
*   **Spin-Down behavior**: Render's free Web Services will "spin down" (go to sleep) if they do not receive any traffic for 15 minutes. When someone visits your URL again, it will take about **50 seconds** to wake up and load the page. This is normal and expected on free tiers.
*   **Database Uptime**: The Clever Cloud MySQL database runs **24/7 without sleeping**, meaning your data is always instantly responsive the moment the web app wakes up!
